# 福利厚生サービス検索の利便性向上ソリューション提案

## 1. 現状の課題分析

貴会の「ライフサポートサービス」は、**お祝い金、健康・医療・福祉、住まい**など、多岐にわたる11のカテゴリに分類されており、非常に充実しています。しかし、ユーザー様から「自分で探すのが大変」というご意見があった通り、以下の課題が確認されました。

| 課題 | 詳細 | ユーザーへの影響 |
| :--- | :--- | :--- |
| **カテゴリ分類の限界** | サービスが「種類」（例：保険、トラベル）で分類されており、「ニーズ」や「ライフステージ」（例：子育て、引っ越し）で横断的に探すことが困難。 | 必要なサービスがどのカテゴリにあるか分からず、見落としが発生しやすい。 |
| **検索機能の不足** | キーワード検索は存在するものの、地域、割引率、利用シーンなど、詳細な条件で絞り込むフィルタリング機能がない。 | 膨大なサービスの中から、自分に最適なものを効率的に見つけられない。 |

## 2. 提案ソリューション：スマート・ナビゲーター機能

この課題を解決するため、既存のサービス分類を活かしつつ、ユーザーの**「目的」**や**「状況」**からサービスにたどり着けるよう、**「スマート・ナビゲーター機能」**の導入を提案します。

### 2.1. サービスへの「タグ付け」導入

各サービスに、ユーザーの具体的なニーズや利用シーンを示す**「タグ」**を付与することを提案します。これにより、カテゴリを横断した検索が可能になります。

| タグの例 | 目的・利用シーン | 該当する可能性のあるカテゴリ（例） |
| :--- | :--- | :--- |
| **#子育て** | 出産、育児、教育費の準備 | お祝い金、住まい（ベビー用品レンタル）、くらしのサポート（速読解Biz） |
| **#節約** | 日常生活のコスト削減 | オンラインショップ、住まい（全厚済でんき、全厚済ガス）、カーライフ（ガソリンスタンド） |
| **#引っ越し** | 転居に伴う手続きや費用 | 住まい（アート引越センター、ふくりの住まい）、カーライフ（カーコンシェルジュ） |
| **#健康維持** | 身体のメンテナンス、病気の予防 | 健康・医療・福祉（健康診断支援、リスク検査）、保険 |
| **#老後の備え** | 終活、相続、介護 | くらしの相談（相続問題相談）、葬儀・お見舞金、住まい（生前整理・遺品整理） |

### 2.2. 新しい検索・フィルタリングインターフェースの設計

現在のキーワード検索に加え、以下の要素を組み合わせた**「複合フィルタリング」**を可能にするインターフェースを導入します。

1.  **タグ検索**: 上記で提案したタグを複数選択して絞り込む機能。
2.  **カテゴリ絞り込み**: 既存の11カテゴリから選択する機能（タグ検索との併用も可能）。
3.  **地域フィルタ**: サービス提供エリア（全国、地域限定など）で絞り込む機能。
4.  **割引・特典タイプ**: 「割引あり」「ポイント付与」「優待券」などで絞り込む機能。

### 2.3. ライフイベント・シナリオ別のおすすめ提案

ユーザーが「結婚が決まった」「子供が生まれた」といった**ライフイベント**を選択するだけで、関連性の高いサービス群を自動で提案する機能です。

| シナリオ名 | 提案されるサービス群（タグに基づく） |
| :--- | :--- |
| **結婚が決まったら** | #結婚、#住まい、#節約、#くらしの相談（離婚問題相談の予防） |
| **マイホームを検討** | #住まい、#引っ越し、#節約（でんき・ガス）、#保険（個人賠償責任） |
| **子供が小学校に入学** | #子育て、#お祝い金（小学校入学祝い）、#教育、#健康維持 |

このシナリオ機能をトップページやマイページに目立つように配置することで、ユーザーは「何から探せばいいか分からない」という状態から解放され、直感的に必要なサービスにたどり着けるようになります。

## 3. 導入効果

このスマート・ナビゲーター機能を導入することで、以下の効果が見込まれます。

1.  **ユーザー満足度の向上**: サービスへのアクセス性が向上し、「探すのが大変」という不満を解消します。
2.  **サービス利用率の向上**: ユーザーの潜在的なニーズに合ったサービスが提案されることで、サービスの利用機会が増加します。
3.  **サイトの価値向上**: 充実したサービス内容を、より分かりやすく、使いやすい形で提供できるようになります。
