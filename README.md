# Prime Smart Navigator (PSN)

福利厚生サービスの検索・提案を革新する、次世代ナビゲーションシステム。

## 概要

**Prime Smart Navigator (PSN)** は、全国福利厚生共済会の福利厚生サービスを、より簡単に、より便利に探せるソリューションです。

### 3つのセールスポイント

1. **AIで予約・連絡まで完結「あなたの秘書ボット」**
   - 面倒な電話確認や予約をAIが代行
   - LINE等のチャットで完結

2. **直感的に見つかる「マルチタグサーチ」**
   - 誰でも使えるシンプルUI
   - 目的・シーンから逆引き検索

3. **導入も運用もかんたん「既存サイト活用型」**
   - 大規模改修不要でスピード導入
   - 既存データベースをそのまま活用

## 機能

### スマート・ナビゲーター
- **マルチタグサーチ**: 「#子育て」「#節約」などのタグを選んで検索
- **カテゴリ検索**: 従来のカテゴリ分けによる検索も可能
- **キーワード検索**: フリーワードでの検索に対応

### AIコンシェルジュ
- **プレゼント相談**: 予算や相手の情報から最適なギフトを提案
- **宿探し**: 条件に合う宿を検索し、予約まで代行
- **トラブル対応**: 緊急時に最寄りのサービスを検索・連絡

## 使用技術

- **フロントエンド**: React 19 + TypeScript + Tailwind CSS 4
- **UI コンポーネント**: shadcn/ui
- **アニメーション**: Framer Motion
- **ルーティング**: Wouter
- **ビルドツール**: Vite

## セットアップ（開発環境）

```bash
# 依存関係のインストール
pnpm install

# 開発サーバーの起動
pnpm dev

# 本番ビルド
pnpm build

# ビルド結果のプレビュー
pnpm preview
```

## プロジェクト構成

```
client/
  ├── public/          # 静的アセット
  │   ├── images/      # 画像ファイル
  │   └── content/     # コンテンツファイル
  ├── src/
  │   ├── pages/       # ページコンポーネント
  │   ├── components/  # 再利用可能なコンポーネント
  │   ├── data/        # データ定義
  │   ├── contexts/    # React Context
  │   ├── hooks/       # カスタムフック
  │   ├── lib/         # ユーティリティ関数
  │   ├── App.tsx      # ルート設定
  │   ├── main.tsx     # エントリーポイント
  │   └── index.css    # グローバルスタイル
  └── index.html       # HTMLテンプレート
```

## デモシナリオ

### マルチタグサーチ
ページ中段の「スマート・ナビゲーター体験デモ」セクションで、実際にタグやカテゴリを選択して、検索結果がリアルタイムに更新される様子を体験できます。

### AIコンシェルジュ
ページ下部の「AIコンシェルジュ・デモ」セクションで、以下のシナリオを試すことができます：
- プレゼント相談
- 宿探し
- 車のトラブル対応
